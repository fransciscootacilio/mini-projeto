# SkillMatch JS

## 📌 Sobre o projeto

O SkillMatch JS é um projeto desenvolvido em JavaScript com o objetivo de análisar a compatibilidade entre um candidato Front-End Júnior e vagas da área de tecnologia.

O sistema compara as habilidades do candidato com os requisitos das vagas e calcula automaticamente o percentual de compatibilidade com cada oportunidade.

Além disso, o projeto também mostra:
- habilidades encontradas;
- habilidades faltantes;
- classificação da compatibilidade;
- vaga com maior aderência;
- recomendações de estudo.
---

# 🚀 Objetivo

O projeto foi criado com foco em praticar os fundamentos do JavaScript moderno e desenvolver uma lógica mais próxima de sistemas utilizados em plataformas de recrutamento e seleção da área de tecnologia.
