# SkillMatch JS

## Sobre o projeto

O SkillMatch JS é um simulador simples de compatibilidade entre um candidato e vagas de Front-End Júnior.

O sistema compara as habilidades do candidato com os requisitos das vagas e mostra:

- percentual de compatibilidade;
- habilidades encontradas;
- habilidades faltantes;
- vaga mais compatível;
- recomendação de estudo.

O projeto foi desenvolvido utilizando JavaScript puro com foco na prática dos principais conceitos estudados no módulo.

---

# Objetivo

O objetivo do SkillMatch JS é simular uma etapa inicial de triagem técnica utilizada por empresas de tecnologia no processo de recrutamento de desenvolvedores Front-End.

A aplicação analisa automaticamente a compatibilidade entre o perfil técnico do candidato e os requisitos das vagas, calculando o percentual de aderência e identificando habilidades que ainda precisam ser desenvolvidas.

Além disso, o projeto busca praticar conceitos fundamentais do JavaScript moderno e simular comportamentos encontrados em sistemas reais de recrutamento e seleção.

# Classificação de compatibilidade

| Percentual | Classificação |
|---|---|
| 80% a 100% | Alta Compatibilidade |
| 50% a 79% | Média Compatibilidade |
| 0% a 49% | Baixa Compatibilidade |
